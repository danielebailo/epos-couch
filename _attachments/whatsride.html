WHAT's R.I.D.E.?

R.I.D.E. is an acronym which stands for Research Infrastrucure Database for EPOS.

This database is a Database of Research Infrastructure participating to EPOS European Project, containing information about Resaerch Infrastructures but NOT scientific data (waveforms, images etc.)

Its main goals are:

Goal 1: show the contents of the EPOS integration plan to all stakeholders (data providers, users from scientific communities, governmental and funding agencies, as well as other stakeholders, privates and industry)

Goal 2: RIDE can represent in the next future an appropriate tool to provide information to the discovery layer forming the new metadata structure envisioned in WG7

Goal 3: RIDE as a tool of the EPOS Core Services. When the EPOS core services will be operational,

RIDE will represent a descriptive database to disseminate and facilitate the use of data infrastructures, data products, etc

Goal 4: Integration of technical and financial/legal questionnaires. RIDE is a powerful tool to integrate collected information in an electronic database and to further collect new information. This integration is needed in order to harmonize the work of different WPs in EPOS PP

Goal 5: For communities (i.e. workign groups) with an advanced infrastructure integration, it can facilitate the dissemination of existing data infrastructures, it can promote a discussion within the community to implement the present data infrastructures , it can be a powerful outreach tool also for mature RIs.
